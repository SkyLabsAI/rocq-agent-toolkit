# RAT Public Dashboard

A web dashboard for the Rocq Agent Toolkit (RAT) built with Next.js, TypeScript, and Tailwind CSS. This dashboard provides an interface for viewing and comparing agent runs, benchmarks, and task details.

## How to Run

### Prerequisites
- Node.js 18.x LTS or newer
- pnpm (install with: `npm install -g pnpm`)

### Development
```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev
```

The dashboard will be available at http://localhost:3000

### Production
```bash
# Build for production
pnpm run build

# Start production server
pnpm start
```
